درباره‌ی گمنام خاموش فراری

این پروژه، پناهگاهی دیجیتال برای واژه‌ها است؛ جایی که اندیشه‌های گم‌شده و صدای خاموشان فرصتی برای بیان می‌یابند.

«گمنام خاموش فراری» تلاشی‌ست برای خلق فضایی آرام، شاعرانه، و انسانی در دل هیاهوی دیجیتال امروز.

این فضا توسط GitHub Pages میزبانی می‌شود و با HTML و CSS ساده طراحی شده است.
