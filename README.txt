# Visual Quote Tool (Persian Watermark Generator)

این ابزار ساده به شما اجازه می‌دهد تا واترمارک فارسی را با انتخاب فونت، رنگ، موقعیت و شفافیت روی تصاویرتان اعمال کرده و ذخیره کنید.

## فایل‌ها:
- `index.html` - صفحه اصلی ابزار
- `style.css` - طراحی ظاهری
- `script.js` - منطق عملکردی

ساخته‌شده توسط گم‌نام خاموش فراری ✨
