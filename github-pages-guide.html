📘 راهنمای گام‌به‌گام انتشار سایت در GitHub Pages

1️⃣ ساخت ریپازیتوری مخصوص GitHub Pages

  • وارد GitHub شوید.
  • روی New repository کلیک کنید.
  • نام ریپازیتوری را وارد کنید: khamoosh-2025.github.io
  • ریپازیتوری را Public انتخاب کنید.
  • روی Create repository کلیک کنید.

2️⃣ آپلود فایل‌های سایت

  • روی Add file → Upload files کلیک کنید.
  • فایل‌های استخراج‌شده از khamoosh-final-github-package.zip را آپلود کنید.
  • در پایین صفحه، برای Commit پیغام بنویسید:
    • 🎉 Finalize GitHub Deployment Package for Dashboard & Tools
  • روی Commit changes کلیک کنید.

3️⃣ فعال‌سازی GitHub Pages

  • به تب Settings → Pages بروید.
  • در قسمت Source، Branch را روی main و مسیر را روی /(root) بگذارید.
  • روی Save کلیک کنید.
